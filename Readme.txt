Набор скриптов предназначенный для регистрации действий пользователя удалённо работающего по RDP.
Основная идея:
Для регистрации действий удалённого пользователя на специально выбранном, промежуточном сервере устанавливаем набор специальных скриптов.
1. Удалённый пользователь заходит на промежуточный сервер под известным ему логином и паролем.
2. Срабатывает logon скрипт
3. Пользователь выбирает один из доступных ему серверов из меню logon скрипта. В текущей версии данный функционал НЕ реализован
4. logon скрипт осуществляет запускает MSTSC на учётную запись соответствующую выбранному серверу.

Практическая проверка показала неработоспособность текущей версии, и возможно самой идеи.
